# Student Depression Prediction using Machine Learning
Machine Learning project to predict student depression using Decision Tree, KNN, and Random Forest

This project predicts student depression using supervised machine learning techniques.

## Models Used
- Decision Tree
- K-Nearest Neighbors (KNN)
- Random Forest

## Dataset
- Source: OpenML (Student Depression Dataset)

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Model training and evaluation
- Performance comparison
- Visualizations (Heatmap, ROC, Confusion Matrix)

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Results
Random Forest achieved the highest accuracy among the models.
